План видов характеристик «Измеряемые показатели объектов ремонта»

Форма предназначена для хранения видов измеряемых показателей объектов ремонта.

В форме выводятся следующие поля:

  • Код — код измеряемого показателя в системе.
  • Наименование — название измеряемого показателя.
  • Тип значения — тип значения измеряемого показателя.
  • Критический максимум — максимальное критическое значение контролируемого показателя, при превышении которого необходим внеплановый ремонт оборудования для предотвращения его аварийного выхода из строя.
  • Допустимый максимум — максимальное допустимое значение контролируемого показателя.
  • Допустимый минимум — минимальное допустимое значение контролируемого показателя.
  • Критический минимум — минимальное критическое значение контролируемого показателя. В случае, если значение контролируемого показателя ниже критического минимума, необходим внеплановый ремонт оборудования для предотвращения его аварийного выхода из строя.
  • Единица измерения — единица измерения измеряемого показателя.
