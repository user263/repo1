Обработка «Общий план работ подразделения»

Обработка предназначена для вывода информации в виде диаграммы или таблицы по видам работ или объектам ремонта для подразделения. Устанавливается период отображения и типы оснований, по которым необходимо вывести информацию. Диаграмма и таблица формируется при нажатии кнопки Построить/Обновить. Итогом служит план работ подразделения с датами.

В форме выводятся следующие поля:

  • Выводить информацию по - выбор оснований для ремонтов.
  • Подразделения - подразделения, для которых составляется общий план работ.
  • Период отображения - период, по которому будут отображаться работы.
  • Выводить данные по - выбор объектов вывода информации: по Видам работ, по Объектам ремота.

 

Закладка «Таблица ремонтов»

  • Подразделение/Вид ремонта - название подразделения, для которого построен план и вид ремонта
  • Дата начала - дата начала ремонта.
  • Дата окончания - дата окончания ремонта.
  • Объект ремонта - название объекта ремонта.
  • Документ-регистратор - дата создания, тип и название документа-основания для ремонта.
