Обработка «Журнал уведомлений пользователя»

Обработка предназначена для вывода всех уведомлений, выполненных в системе.

В форме выводятся следующие поля:

  • Только непрочитанные - признак отбора только непрочитанных уведомлений.
  • Спсоб уведомления - список способов уведомлений: Всплывающее уведомление, По почте, Все.
  • Дата начала - дата начала при отборе по дате и времени.
  • Дата окончания - дата окончания при отборе по дате и времени.
  • Документ- название документа, для которого создано уведомление.
  • Тема письма - тема письма-уведомления.
  • Дата уведомления - дата создания уведомления.
  • Текст уведомления - полный текст уведомления.
