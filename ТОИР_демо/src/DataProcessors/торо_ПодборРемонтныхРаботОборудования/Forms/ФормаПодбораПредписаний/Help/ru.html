Обработка «Подбор ремонтных работ оборудования»

Режим вызывается при нажатии кнопки [Подбор], которая располагается в формах документов системы всегда в командной панели над табличной частью, содержащей перечень ремонтов оборудования.

Режим предназначен для оптимизации заполнения пользователем документов, вводящихся на основании других документов.

 

В форме выводятся следующие поля:

  • Верхнее табличное поле - документ - основание.
  • Среднее табличное поле – ремонты оборудования, содержание одноименной части документа - основания (то чем будет заполнен текущий документ).
  • Нижнее табличное поле (может отсутствовать) – ремонтные работы документа-основания.
  • Период - накладывает отбор по дате на записи табличного поля ремонты оборудования.

Выбор объектов ремонта выполняется с помощью кнопки [Выбрать].

Используется для следующих видов документов:

 

Документ "Акт о выполнении этапа работ"

Документ "Дефекты оборудования"

Документ "Смета ремонта (Заявка на ремонт)"

Документ "Наряд на выполнение ремонтных работ"

Документ "Оценка работы подрядчиков"

Документ "План-график ППР"

Документ "Предписания"
