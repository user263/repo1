Обработка «Корректировка ППР по РМ»

Обработка предназначена для анализа рентабельности плановых ремонтов и их корректировки в зависимости от суммы установленных лимитов на ремонты.

 

В форме выводятся следующие поля:

  • Организация, Направление, Год - поля, устанавливающие отбор на плановые ремонты.
  • Табличное поле - поле корректировки плановых ремонтов ППР.
  • Сумма установленных лимитов - сумма доступных для плановых ремонтов денежных средств, установленная на выбранный год.
  • Общая оставшаяся сумма ремонтов - сумма денежных средств, необходимая для выполнения всех ремонтов, не отмеченных галкой "Удалить".
