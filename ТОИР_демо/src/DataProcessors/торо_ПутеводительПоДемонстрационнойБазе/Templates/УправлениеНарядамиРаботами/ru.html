6. Управление нарядами и работами

Чтобы реализовать учет выполнения работ, запланированных в рамках оперативного управления работами по ТО и ремонтам, по оформленным ранее заявкам на ремонт сформированы наряды на выполнение ремонтных работ. По завершении работ наряды закрыты актами, в которых указаны материальные затраты, трудозатраты, исполнители ремонтных работ.

6.1. Оформление заявок на ремонты

Для оформления заявок в системе предназначен документ «Смета ремонта (Заявка на ремонт)».

При вводе заявки на ремонт в системе необходимо указать следующие основные данные:

  • документ-основание создания заявки: обнаруженный дефект или график ППР;
  • объект ремонта, на ремонт которого оформляется заявка;
  • вид ремонта: аварийный при создании заявки на основании дефекта или выбранный пользователем из графика ППР;
  • даты начала и окончания периода выполнения ремонтных работ по заявке;
  • перечень ремонтных работ, которые необходимо выполнить по заявке с указание материальных затрат и трудозатрат;

Документ «Смета ремонта (Заявка на ремонт)» №РК000000002 от 01.02.2014  9:00:00.

Документ «Смета ремонта (Заявка на ремонт)» №РК000000014 от 10.02.2014  9:00:00.

6.2. Формирование и контроль нарядов на выполнение ремонтных работ

Для оформления нарядов в системе предназначен документ «Наряд на выполнение ремонтных работ».

При вводе наряда на ремонт в системе необходимо указать следующие основные данные:

  • документ-основание создания наряда;
  • объект ремонта, на ремонт которого оформляется наряд;
  • даты начала и окончания периода выполнения ремонтных работ по наряду;
  • перечень ремонтных работ, которые необходимо выполнить по наряду;

Документ «Наряд на выполнение ремонтных работ» №РК000000001 от 01.02.2014  9:00:00.

6.3. Учет выполненных работ

Учет выполненных работ в системе оформляется документом «Акт о выполнении этапа работ».

При вводе акта на ремонт в системе необходимо указать следующие основные данные:

  • документ-основание создания акта;
  • объект ремонта, на ремонт которого оформляется акт;
  • фактические даты начала и окончания ремонтных работ по акту;
  • перечень ремонтных работ, которые выполнены по акту с указание материальных затрат и трудозатрат;

Документ «Акт о выполнении этапа работ» №РК000000001 от 01.03.2014  9:00:00.
