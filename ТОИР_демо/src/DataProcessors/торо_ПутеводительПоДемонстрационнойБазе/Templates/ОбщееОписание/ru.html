Путеводитель по демонстрационной базе поможет Вам оперативно  познакомиться с работой системы в качестве пользователя, получить общее представление о логике демонстрационной базы и системы в целом.

Путеводитель по демонстрационной базе содержит ссылки на документы, введенные в демонстрационную базу по бизнес-процессам ТОиР. Можно открыть документ прямо из путеводителя и проанализировать его.

Список всех разделов путеводителя отображается в списке разделов в левой части окна. Для перехода к любому разделу достаточно кликнуть мышкой по его названию. Разделы сгруппированы по бизнес-процессам.

В демонстрационной базе отражены основные операции с примерами формирования документов, сопровождающих ремонтные работы.

Данный путеводитель предназначен для самостоятельного начального ознакомления с основным базовым функционалом системы.

Для более подробного изучения возможностей 1С:ТОиР, рекомендуем Вам воспользоваться электронным мультимедийным курсом  «1С:Электронное обучение. ТОиР шаг за шагом» (код продукта в прайс-листе фирмы 1С – 4601546111081), который Вы легко сможете приобрести у любого партнера фирмы 1С.
