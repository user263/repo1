Обработка «Подбор объектов ремонтных работ»

Режим вызывается при нажатии кнопки [Подбор], которая располагается в формах документов системы всегда в командной панели над табличной частью, содержащей перечень оборудования.

Режим предназначен для оптимизации заполнения пользователем сведений о наработке объектов ремонта, данные по которым зарегистрированы в системе. Верхняя часть формы содержит иерархический список объектов ремонта.

 

В форме выводятся следующие поля:

  • Структура иерархии - вид иерархии объектов ремонта.
  • Наименование – название объекта ремонта.
  • Не учитывать подчинённые ОР - при выборе объекта ремонта в табличную часть будет попадать только головной объект ремонта 
  • Учитывать только подчинённые ОР 1-го уровня - при выборе объекта ремонта в табличную часть будут попадать только его подчинённые объекты первого уровня.
  • Учитывать подчинённые ОР 1-го уровня - при выборе объекта ремонта в табличную часть попадёт сам объект ремонта и его подчинённые объекты первого уровня.
  • Учитывать подчинённые ОР всех уровней - при выборе объекта ремонта в табличную часть попадёт сам объект ремонта его подчинённые объекты всех уровней.

 

Выбор объектов ремонта выполняется с помощью кнопки [Выбрать].

Форма обработки в зависимости от документа, в котором она вызывается, может иметь ещё ряд полей:

  • Запрашивать вид параметров наработки - установка флага вызывает форму подбора параметра наработки при выборе объекта ремонта.
  • Проставлять плановую наработку - при установленном флаге в поле Наработка будет проставляться плановая наработка за определённый период.
  • Учитывать простои ОР - при установленном флаге будет учитываться время простоя.
  • Запрашивать контролируемые показатели - установка флага вызывает форму подбора контролируемого показателя при выборе объекта ремонта.
