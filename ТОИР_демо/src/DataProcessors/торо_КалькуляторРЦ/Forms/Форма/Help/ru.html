Обработка «Калькулятор ремонтного цикла»

Обработка «Калькулятор ремонтных нормативов» предназначена для проведения анализа «что-если».

Данный вид анализа позволяет смоделировать любой ремонтный цикл или план-график ППР без занесения данных в действующую ИБ и отслеживать результаты изменения нормативов на план-график ППР и ремонтный цикл.

Калькулятор ремонтных нормативов позволяет:

1. Использовать в расчетах нормативы как хранящиеся в ИБ, так и заданные пользователем.

2. Рассчитывать план-график ППР и ремонтный цикл.

3. Быстро перемещаться между нормативами и результатми их изменения.

4. Записывать измененные нормативы в ИБ.

Обработка имеет четыре закладки:

1. Нормативы

    • Закладка "Нормативы" предназначена для формирования списка  видов и объектов ремонта, загрузки, редактирования и сохранения нормативов.
    • Для заполнения списка видов ремонта предназначены кнопки "Добавить ОР", "Добавить ВР" и "Подбор".
    • Следующим шагом после добавления необходимых видов ремонта в список, является заполнение ремонтных нормативов. При заполнении могут использоваться два варианта: заполнение нормативов "вручную" и загрузка нормативов из ИБ(Кнопка "Получить нормативы").
    • Для быстрого переключения между нормативами вида ремонта и результатами их изменений предназначены кнопки, вызываемые из контекстного меню ("Проверить результаты...").
    • Завершающим этапом работы обработки является занесение нормативов в ИБ. Для этого используется кнопка "Записать нормативы". Окно, появляющееся перед записью, предназначено для выбора режима записи нормативов (записать для выбранного объекта ремонта или для типового объекта ремонта). По умолчанию флажки установлены в соответствии с тем, откуда норматив был загружен(из данных объекта ремонта, из данных типового объекта ремонта или введён пользователем самостоятельно).

2. План-график ППР

Закладка "План-график ППР" предназначена для расчета и анализа план-графика ППР. На закладке имеются поля, устанавливающие периодичность расчета. При нажатии кнопки "Рассчитать" происходит расчет план-графика с использованием нормативов, установленных на закладке "Нормативы". Список видов ремонта может использоваться для навигации по план-графику. Контекстное меню списка ремонтов позволяет быстро перейти к редактированию нормативов выбранного вида ремонта.

3. Диаграмма Ганта

Закладка "Диаграмма Ганта" предназначена для работы с диграммой Ганта, построенной по план-графику ППР.

4. Ремонтный цикл

Закладка "Ремонтный цикл" предназначена для  работы с мнемосхемой ремонтного цикла. Для построения мнемосхемы необходимо выбрать объект и вид ремонта, для которых будет рассчиатн ремонтный цикл. При построении мнемосхемы, также заполняется таблица с данными по каждому ремонту. Навигация по таблице может осуществляться путем выбора ремонта на мнемосхеме. Кнопка "Перейти к нормативам" позволяет перейти к редактированию нормативов.
