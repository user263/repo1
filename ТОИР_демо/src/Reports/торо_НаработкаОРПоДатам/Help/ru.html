Отчет «Наработка оборудования на дату планового ремонта (ТО)»

Отчет предназначен для получения информации о наработке объектов ремонта на дату запланированного ремонта. Под плановым ремонтом (ТО) понимается ремонт, запланированный документом "План-график ППР". Для доступа к отчету необходимо установить настройку системы в разделе Настройка параметров подсистемы ТОиР - Интерфейс и ввод документов - Учет показателей эксплуатации: Учет наработки оборудования.
Открывается по команде Наработка оборудования на дату планового ремонта (ТО):

  • в списке Отчеты подсистемы Учет показателей эксплуатации;
  • в списке Отчеты подсистемы Планирование ТОиР.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. b. Для отбора плановых ремонтов, на дату которых считается наработка, в отчете можно отредактировать параметр отчета: Период отчета. При этом отбор будет выполняться по дате начала ремонта.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один предопределенный вариант отчета - Наработка оборудования на дату планового ремонта (ТО): таблица.
Данные о значениях показателей наработки сгруппированы по объектам ремонта и видам ремонтных работ.
В строках выводится информация о датах запланированных в системе ремонтов и соответствующих этим датам значениям наработки. Планирование ремонтов объектов ремонта выполняется документами "План-график ППР", а значение наработки рассчитывается по формуле, представленной в описании поля:

  • Объект ремонта - наименование объекта ремонта;
  • Вид ремонтных работ - вид ремонта, для которого запланирован период выполнения (для этого объекта ремонта зарегистрирован документ "План-график ППР");
  • Дата начала - дата начала запланированного ремонта из документа "План-график ППР";
  • Ед. Изм. - единица измерения показателя наработки;
  • Наработка - значение показателя наработки объекта ремонта на дату запланированного ремонта, рассчитывается от даты ввода в эксплуатацию объекта ремонта (поле "Дата ввода в эксплуатацию" в карточке объекта ремонта) до даты начала запланированного ремонта (поле "Дата начала ремонтных работ" в документе "План-график ППР") с учетом периода наработки (поля "Количество периодов" и "Период" на закладке "Наработка" в карточке объекта ремонта), графика работы (поле "График работы" на закладке "Общие" в карточке объекта ремонта), КИО (поле "КИО" на закладке "Наработка" в карточке объекта ремонта), информации о простоях объекта ремонта (для дат, находящихся в указанном интервале и имеющих простои - т.е. зарегистрирован документ "Состояния объектов ремонта" со значением поля "Вид эксплуатации" = "Простой" - расчет наработки не осуществляется) и фактически зарегистрированной наработки за этот период (поле "Наработка"/ "Значение" из документов "Учет наработки оборудования", "Установка произвольного значения наработки"/ "Ввод начальных данных").
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
