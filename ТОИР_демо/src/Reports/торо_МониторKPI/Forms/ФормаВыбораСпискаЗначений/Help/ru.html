1. Начальная настройка системы для работы монитора показателей (KPI)

Для работы с Монитором показателей (KPI) необходимо заполнить справочники «Показатели процессов (KPI)» и «Варианты анализа показателей». Можно воспользоваться поставляемой моделью показателей (первая версия модели была загружена в систему автоматически, в дальнейшем для обновления поставляемой модели необходимо использовать команду «Восстановить поставляемую модель показателей» в форме списка справочника «Показатели процессов (KPI)»), или создать собственные показатели.

Если используются показатели с вариантом заполнения «Внешний показатель», то для расчета их значений необходимо ввести документы «Регистрация значений внешних показателей».

Если в вариантах анализа показателей установлены флаги «Выполнить нормализацию» или планируется использование интегральной оценки в мониторе показателей, то необходимо ввести документы «Установка целевых значений показателей (KPI)».

2. Использование монитора показателей (KPI)

В мониторе предусмотрено два режима просмотра:

  • Обычный. В этом режиме значение каждого показателя выводится отдельно от остальных в сетке, количество колонок сетки можно регулировать.
  • Интегральная оценка. В этом режиме значения всех показателей нормируются и выводятся в одну радиальную диаграмму. Можно выбрать, что является сериями диаграммы: разные периоды или разные значения аналитик. Для вывода диаграммы необходимо выбрать хотя бы 3 варианта анализа, различающихся по совокупности значений реквизитов «Показатель», «Значение», «Агрегация данных», «Выбрать сортировку».

Ключевые настройки монитора – «Тип показателей» и «Показатели (KPI)». В настройке «Показатели (KPI)» задается список элементов справочника «Варианты анализа показателей». В настройке «Тип показателей» задается отбор по типу показателей, который действует следующим образом:

  • Для значения «Показатели объекта ремонта» доступны только показатели объекта ремонта. Аналитикой, по которой выполняется сворачивание таблиц данных показателей, является объект ремонта.
  • Для значения «Показатели подразделения» доступны показатели подразделения и показатели объекта ремонта. Аналитикой, по которой выполняется сворачивание таблиц данных показателей, является подразделение. Для сворачивания таблиц данных от показателей объектов ремонта до уровня подразделений используется агрегатная функция, указанная в реквизите «Агрегация данных» варианта анализа.
  • Для значения «Показатели организации» доступны все показатели. Аналитикой, по которой выполняется сворачивание таблиц данных показателей, является организация. Для сворачивания таблиц данных от показателей более низкого уровня до уровня организаций используется агрегатная функция, указанная в реквизите «Агрегация данных» варианта анализа.

Остальные настройки монитора расположены в меню «Еще / Настройки…».

  • Количество колонок показателя – количество колонок в мониторе.
  • Период - обязательный параметр, период, за который собираются данные.
  • Детализация по периоду – периодичность получения данных для типов анализа «Динамика изменения» и «Покомпонентное сравнение (динамика)».
  • Период сравнения – период сравнения для типа анализа «Сравнение с прошлым периодом».
  • Организация, Подразделение, Объект ремонта – отборы по аналитикам.

В режиме просмотра «Интегральная оценка» вид диаграммы определяется настройкой «Тип серий интегральной оценки»:

  • Сравнение с прошлым периодом – сериями диаграммы являются текущий период и период сравнения.
  • По заданной периодичности – сериями диаграммы являются периоды с учетом указанной в настройках периодичности.
  • По объектам анализа – сериями диаграммы являются объекты ремонта, подразделения или организации.

Доступно сохранение и восстановление настроек с помощью команд «Сохранить настройки…» и «Выбрать настройки…».

Все диаграммы можно вывести на печать. Вид печатной формы зависит от текущего режима просмотра.

Отличие между кнопками «Сформировать» и «Перерисовать» в том, что кнопка «Сформировать» полностью переформирует отчет, заново получая данные показателей, что может занять длительное время. А кнопка «Перерисовать» использует полученные ранее при формировании отчета данные, и только заново выводит их на экран (можно использовать после изменения настроек в варианте анализа, изменения количества колонок монитора или изменения типа серий интегральной оценки).
