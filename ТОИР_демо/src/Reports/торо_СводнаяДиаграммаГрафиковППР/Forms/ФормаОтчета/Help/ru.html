Отчет «Сводная диаграмма графиков ППР»

Отчет предназначен для анализа планово-предупредительных ремонтов на указанный период в виде диаграммы Ганта графиков ППР. Информация представлена по месяцам.

Поля отчета:

  • Объект ремонта - объект ремонта, по которому будет сформирован отчет.
  • Период с ... по ... - период, на который будет построена диаграмма ППР.
  • Вид ремонта - интересующий вид ремонта для формирования отчета.
  • Подразделение - подразделение, по которому необходимо сформировать графики ППР.
  • Отображать иерархию - признак, позволяющий отобразить полную иерархию объектов ремонта.
