Отчет «Контролируемые показатели»

Отчет предназначен для получения информации об изменениях значений контролируемых показателей объектов ремонта за период. Для доступа к отчету необходимо установить настройку системы в разделе Настройка параметров подсистемы ТОиР - Интерфейс и ввод документов - Учет показателей эксплуатации: Учет контролируемых показателей.
Открывается по команде Контролируемые показатели: в списке Отчеты подсистемы Учет показателей эксплуатации.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. Для фильтра по периоду регистрации контролируемых показателей в отчете можно отредактировать параметр отчета: Период отчета.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует два предопределенных варианта отчета:

  1. Контролируемые показатели: таблица - открывается по умолчанию. Данные о значениях контролируемых показателей сгруппированы по объектам ремонта и контролируемым показателям.
    В этом варианте отчета по умолчанию в строках выводится информация о зарегистрированных в системе значениях контролируемых показателей. Регистрация значений контролируемых показателей выполняется документами "Учет контролируемых показателей" и "Ввод начальных данных":
    • Объект ремонта - наименование объекта ремонта;
    • Показатель - наименование контролируемого показателя объекта ремонта;
    • Ед. - единица измерения контролируемого показателя;
    • Документ - дата и время регистрации контролируемого показателя;
    • Значение - значение контролируемого показателя объекта ремонта.
      В зависимости от близости значений к допустимым границам, заданным для контролируемого показателя в плане видов характеристик «Контролируемые показатели объектов ремонта» или измененными в справочниках «Объекты ремонта» и «Типовые объекты ремонта», они отображаются различными цветами:
      • красный - значение находится за пределами критического максимума или критического минимума;
      • желтый - значение находится за пределами допустимого максимума или допустимого минимума, но в пределах критических значений контролируемого показателя;
      • зеленый - значение находится в пределах допустимых границ, т.е. между допустимым максимумом и допустимым минимумом.
    Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
  2. Контролируемые показатели: диаграмма - отчет предназначен для получения информации о значениях контролируемых показателей объектов ремонта в разрезе объекта ремонта и контролируемого показателя в графическом виде.
