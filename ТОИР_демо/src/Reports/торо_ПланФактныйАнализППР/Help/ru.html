Отчет «План-фактный анализ ППР»

Отчет предназначен для план-фактного анализа выполнения работ, рассчитанных в графиках ППР. Синим цветом закрашиваются ячейки таблицы для запланированных работ по техническому обслуживанию и ремонтам, красным – для работ, которые выполнены по факту исполнителями, зелёным - даты простоя оборудования (доступно при установленном признаке Отображать простои оборудования).

Настройки отчёта:

  • Период с - начальная дата периода отображения данных.
  • по - конечная дата периода отображения данных.
  • Метод разбиения - в разрезе которой в каждой колонке отчета будет представлена информация.
  • Отображать простои оборудования - при установленном признаке в отчёте появится строка, обозначающая простои оборудования.
  • Вид простоя - наименование вида простоя. Вбирается из списка: Аварийные ремонтные работы, Без работы, Плановые ремонтные работы, Прочее, Эксплуатация.
  • Причина простоя - наименование причины простоя оборудования.
  • Не отображать простои без плановых ремонтов - при установленном признаке будут отбражаться простои только по плановым ремонтам.
  • Документы ППР - наименования одного или нескольких план-графиков ППР, по которым будет сформирован отчёт.
