Отчет «Состояние оборудования»

Отчет предназначен для получения информации об изменениях значений контролируемых показателей объектов ремонта за период в графическом виде. Для доступа к отчету необходимо установить настройку системы в разделе Настройка параметров подсистемы ТОиР - Интерфейс и ввод документов - Учет показателей эксплуатации: Учет контролируемых показателей.
Также с помощью отчета можно одновременно получить информацию о состоянии объектов ремонта и значениях наработки объектов ремонта на эти даты в графическом виде. Для работы с наработкой требуется включить настройку системы в разделе Настройка параметров подсистемы ТОиР - Интерфейс и ввод документов - Учет показателей эксплуатации: Учет наработки оборудования.
Открывается по команде Состояние оборудования: из карточки объекта ремонта на вкладке Показатели закладки Показатели эксплуатации.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки.
Для фильтра по выводимым данным в отчете можно отредактировать параметры отчета:

  • Объект ремонта - отбор по объектам ремонта;
  • Период выполн - отбор по дате регистрации значений показателей (и контролируемых, и наработки);
  • Контролируемые показатели - отбор по контролируемым показателям объекта ремонта;
  • Показатель наработки* - отбор по показателям наработки объекта ремонта, значения по которому будут выводиться в отчет.
По умолчанию параметры отчета заполняются значениями из карточки объекта ремонта:
  • Объект ремонта - текущий объект ремонта;
  • Период выполн - текущий год;
  • Контролируемые показатели - выбранный контролируемый показатель;
  • Показатель наработки* - <Пустое значение>.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один предопределенный варианта отчета - Состояние оборудования. Данные о значениях контролируемых показателей и показателей наработки представлены в графическом виде, т.е. для каждого контролируемого показателя или показателя наработки сформирован отдельный график со значениями за период.
Регистрация значений контролируемых показателей выполняется документами "Учет контролируемых показателей" и "Ввод начальных данных", а регистрация значений показателей наработки - документами "Учет наработки оборудования", "Установка произвольного значения наработки" и "Ввод начальных данных".
В графике контролируемых показателей дополнительно для значения измеряемого показателя отображаются 5 направляющих линий, соответствующих допустимым границам показателя. Расшифровка цветов приведена в легенде, настройки допустимых границ показателя выполняются в карточке «Контролируемые показатели объектов ремонта» и могут быть изменены в справочниках «Объекты ремонта» и «Типовые объекты ремонта». График показателей наработки строится по значениям наработки, зарегистрированным в системе.

Примечание*:
Если в настройках системы включен флаг "Учет наработки оборудования", то параметр "Показатель наработки" отображается.
