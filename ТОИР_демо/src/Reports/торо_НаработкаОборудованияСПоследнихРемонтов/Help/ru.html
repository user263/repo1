Отчет «Наработка оборудования с последних ремонтов»

Отчет предназначен для получения информации о наработке объектов ремонта на текущий момент времени с даты окончания последнего завершенного ремонта. Под завершенным ремонтом понимается ремонт, закрытый документом "Акт о выполнении этапа работ" с флагом "Завершить ремонт". Для доступа к отчету необходимо установить настройку системы в разделе Настройка параметров подсистемы ТОиР - Интерфейс и ввод документов - Учет показателей эксплуатации: Учет наработки оборудования.
Открывается по пиктограмме Открыть отчет "Наработка оборудования с последних ремонтов": из карточки объекта ремонта на вкладке Наработка закладки Показатели эксплуатации.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки.
В отчете можно настроить отборы путем установки значений параметров отчета:

  • Период отчета - отбор по дате окончания ремонтов;
  • Объект ремонта - отбор по объектам ремонта;
  • Структура иерархии - отбор по структуре иерархии. Структура иерархии определяет список подчиненных объектов ремонта при отборе (значение параметра имеет смысл только при включении флага "Вместе с подчиненными");
  • Флаг "Вместе с подчиненными" - отбор по объектам ремонта, подчиненных данному (проверяется не только непосредственное подчинение объекту ремонта, но и через 1, 2 и более уровни подчиненности).
По умолчанию при открытии отчета из карточки объекта ремонта в параметрах отчета заполнено значение "Объект ремонта".

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует один предопределенный вариант отчета - Наработка оборудования с последних ремонтов: таблица. Данные о значениях показателей наработки сгруппированы по объектам ремонта и видам ремонтных работ.
В отчете по умолчанию в строках выводится информация о датах последних выполненных ремонтов объектов ремонта и значениях наработки с этих дат. Выполнение ремонтов объектов ремонта регистрируется документами "Акт о выполнении этапа работ" (с флагом "Завершить ремонт"), а значение наработки рассчитывается по формуле, представленной в описании поля:

  • Объект ремонтных работ - наименование объекта ремонта;
  • Последний ремонт - вид ремонта, у которого дата окончания (поле "Дата окончания" из документа "Акт о выполнении этапа работ" с флагом "Завершить ремонт") последняя для данного объекта ремонта;
  • Дата окончания ремонта - дата окончания последнего выполненного ремонта (поле "Дата окончания" из документа "Акт о выполнении этапа работ" с флагом "Завершить ремонт");
  • Наработка с последнего ремонта - значение показателя наработки объекта ремонта на текущий момент времени с даты окончания последнего ремонта, рассчитывается от даты окончания последнего выполненного ремонта (поле "Дата окончания" из документа "Акт о выполнении этапа работ" с флагом "Завершить ремонт") до текущего момента времени с учетом периода наработки (поля "Количество периодов" и "Период" на закладке "Наработка" в карточке объекта ремонта), графика работы (поле "График работы" на закладке "Общие" в карточке объекта ремонта), КИО (поле "КИО" на закладке "Наработка" в карточке объекта ремонта), информации о простоях объекта ремонта (для дат, находящихся в указанном интервале и имеющих простои - т.е. зарегистрирован документ "Состояния объектов ремонта" со значением поля "Вид эксплуатации" = "Простой" - расчет наработки не осуществляется) и фактически зарегистрированной наработки за этот период (поле "Наработка"/ "Значение" из документов "Учет наработки оборудования", "Установка произвольного значения наработки"/ "Ввод начальных данных"). Если фактическая наработка зафиксирована на дату позже текущей – эта наработка также выводится в отчет, т.е. выводится информация до большей из двух дат: текущей даты или даты, по которую зафиксирована наработка;
  • Показатель.Единица измерения - единица измерения показателя наработки.
Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
