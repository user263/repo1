Отчет «Наработка оборудования»

Отчет предназначен для получения информации об изменениях значений наработки объектов ремонта за период. Доступен при включенном флаге в разделе Настройка параметров подсистемы ТОиР - Интерфейс и ввод документов - Учет показателей эксплуатации: Учет наработки оборудования.
Открывается по команде Наработка оборудования: в списке Отчеты подсистемы Учет показателей эксплуатации.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. Для фильтра по периоду регистрации наработки объектов ремонта в отчете можно отредактировать параметр отчета: Период отчета.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует два предопределенных варианта отчета:

  1. Наработка оборудования: таблица - открывается по умолчанию. Данные о значениях показателей наработки сгруппированы по организациям, объектам ремонта и показателям наработки за период.
    В этом варианте отчета по умолчанию в строках выводится информация о зарегистрированных в системе значениях показателей наработки. Регистрация значений показателей наработки выполняется документами "Учет наработки оборудования", "Установка произвольного значения наработки" и "Ввод начальных данных":
    • Организация - организация объекта ремонта;
    • Объект ремонта - наименование объекта ремонта;
    • Показатель наработки, Ед. изм. - наименование показателя наработки объекта ремонта и его единица измерения;
    • Начало периода - дата и время начала регистрации наработки (поле "Дата работы с"/ "Дата последней наработки"/ "Дата ввода в экспл." из документов "Учет наработки оборудования"/ "Установка произвольного значения наработки"/ Карточки объекта ремонта, если регистрация значения наработки вводится впервые для показателя наработки объекта ремонта);
    • Окончание периода - дата и время окончания регистрации наработки (поле "Дата работы по"/ "Дата и время операции" из документов "Учет наработки оборудования", "Ввод начальных данных"/ "Установка произвольного значения наработки");
    • Наработка - значение показателя наработки объекта ремонта (поле "Наработка"/ "Значение" из документов "Учет наработки оборудования", "Установка произвольного значения наработки"/ "Ввод начальных данных").
    Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
  2. Наработка оборудования: диаграмма - отчет предназначен для получения информации о значениях наработки объектов ремонта в разрезе объекта ремонта и показателя наработки в графическом виде.
