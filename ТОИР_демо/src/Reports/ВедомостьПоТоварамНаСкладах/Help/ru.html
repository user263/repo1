Варианты формирования:

Движения товаров по складам

Вариант отображает все движения товаров с группировкой по имеющимся складам за выбранный период.

Остатки товаров на складах

Вариант отображает остатки товаров в разбивке по складам на конечную дату выбранного периода.

Товары в процессе отгрузки

Вариант отображает количество товарных позиций, находящихся в стадии отгрузки (например, оформлен Расходный ордер в статусе К отгрузке), что необходимо при пересчете товаров во время проведения инвентаризации для учета товарных позиций по неотгруженным накладным.
