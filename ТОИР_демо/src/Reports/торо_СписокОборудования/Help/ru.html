Отчет «Список оборудования»

Отчет предназначен для получения информации об объектах ремонта, используемых на предприятии.
Открывается по команде Список оборудования:

  • в списке Отчеты подсистемы Учет оборудования и нормативов;
  • кнопки Перейти в списке справочника "Объекты ремонтов" подсистемы Учет оборудования и нормативов.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. Для фильтра по структуре иерархии объектов ремонта можно отредактировать параметр отчета: Cтруктура иерархии.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует три предопределенных варианта отчета:

  1. Процент износа оборудования: диаграмма - отчет предназначен для получения информации об объектах ремонта в разрезе процента износа каждого в графическом виде. Сегменты диаграммы разделены на 5 групп:
    • износ менее 20%;
    • износ от 20% до 40%;
    • износ от 40% до 60%;
    • износ от 60% до 80%;
    • износ свыше 80%.
  2. Список оборудования: таблица - открывается по умолчанию, в нем объекты ремонта сгруппированы по вышестоящим в иерархии объектам ремонта или группам объектов ремонта, имеющим подчиненные объекты ремонта, т.е. родителям объектов ремонта.
    В этом варианте отчета по умолчанию в строках выводятся поля, которые берутся из карточки объекта ремонта или рассчитываются по формуле, представленной в описании поля:
    • Объект ремонта (иерархия) - объект ремонта или группа объектов ремонта, имеющие подчиненные объекты, т.е. родитель объекта ремонта;
    • Объект ремонта - наименование объекта ремонта;
    • Процент использования - отношение отработанных месяцев объекта ремонта (рассчитывается от даты ввода в эксплуатацию объекта ремонта до текущей даты, т.е. количество завершенных месяцев) к сроку его полезного использования (берется из карточки объекта ремонта), выраженное в процентах;
    • Остаточный срок полезного использования - количество месяцев, которое объект ремонта может отработать с учетом уже отработанного периода времени (рассчитывается от даты ввода в эксплуатацию объекта ремонта до текущей даты, т.е. количество завершенных месяцев) и сроком его полезного использования (берется из карточки объекта ремонта);
    • Инвентарный номер - инвентарный номер объекта ремонта;
    • Технологический номер - технологический номер объекта ремонта;
    • Номер паспорта (регистрационный) - номер паспорта объекта ремонта;
    • Изготовитель - изготовитель объекта ремонта;
    • Заводской номер - заводской номер объекта ремонта;
    • Дата выпуска (постройки) - дата выпуска объекта ремонта;
    • Дата ввода в эксплуатацию - дата ввода в эксплуатацию объекта ремонта;
    • Плановый график работы - график работы объекта ремонта.
    Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
  3. Типы оборудования: диаграмма - отчет предназначен для получения информации об объектах ремонта в разрезе вышестоящих в иерархии объектах ремонта или группах объектов ремонта, имеющих подчиненные объекты ремонта, т.е. родителей объектов ремонта, в графическом виде.
