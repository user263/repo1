Отчет «Затраты номенклатуры»

Отчет предназначен для план-фактного анализа затрат МТО по ремонтным работам. Отчет содержит перечень технологических операций по ремонтам оборудования за указанный период. Для каждого операции указываются запланированные и фактически использованные количества номенклатуры и их суммы.
