Отчет «Внешние основания для работ»

Отчет предназначен для получения информации о зарегистрированных в системе внешних основаниях работ и состоянии их выполнения за период. Для доступа к отчету необходимо установить настройку системы в разделе Настройка параметров подсистемы ТОиР - Интерфейс и ввод документов - Планирование ТОиР: Использовать внешние основания для работ.
Открывается по команде Внешние основания для работ: в списке Отчеты подсистемы Планирование ТОиР.

Параметры отчета

Обращение к режиму настройки отчета выполняется с помощью кнопки Настройки. Для фильтра по плановой дате обработки внешнего основания для работ в отчете можно отредактировать параметр отчета: Период отчета.

Варианты отчета

С помощью кнопки Варианты отчета можно открыть список доступных вариантов отчета. Существует два предопределенных варианта отчета:

  1. Внешние основания для работ: таблица - открывается по умолчанию. Данные о внешних основаниях сгруппированы по объектам ремонта и документам "Внешние основания для работ".
    В этом варианте отчета по умолчанию в строках выводится информация о зарегистрированных в системе внешних основаниях для работ. Регистрация внешних оснований выполняется документами "Внешние основания для работ":
    • Объект ремонта - наименование объекта ремонта;
    • Регистратор внешнего основания - ссылка на документ "Внешние основания для работ";
    • Описание - поле "Описание" из документа "Внешние основания для работ";
    • Плановая дата обработки внешнего основания - поле "Плановая дата ремонта" из документа "Внешние основания для работ";
    • Обработано - одно из двух значений: "Да" или "Нет". Внешние основания считаются обработанными, если на основании них введены документы "Выявленные дефекты".
      В зависимости от значения поля изменяется условное оформление данных о внешнем основании (поля "Описание", "Плановая дата обработки внешнего основания", "Обработано"):
      • При "Да": цвет "черный", начертание "обычное";
      • При "Нет": цвет "красный", начертание "курсив".
    Для изменения структуры данных отчета или настройки отборов необходимо выполнить обращение к режиму настройки отчета с помощью кнопки Настройки.
  2. Внешние основания для работ: диаграмма - отчет предназначен для анализа количественного отношения в графическом виде обработанных и необработанных внешних оснований для работ.
