Отчет «База данных по подрядчикам с историей»

Отчет вызывается из пункта меню «Отчеты»  → «База данных по подрядчикам с историей».

Отчет предназначен для оценок работ, выполненных подрядчиками. В отчете для каждого подрядчика отображаются акты выполненных работ с указанием установленных оценок различных видов.
