Регистр накопления "Фактические материальные затраты ремонтных работ"

Регистр предназначен для хранения информации о фактических затратах номенклатуры по выполненным ремонтным работам.

Движения в регистре происходя при проведение документа "Акт о выполнение этапа ремонтных работ", если в нем перечислены фактические затраты номенклатуры.

Данные регистра используются при формирование отчета "Фактические затраты по объектам ремонта".
