Документ «Установка произвольного значения наработки»

В форме выводятся следующие поля: 

  • Номер — номер документа.
  • от — дата регистрации документа.
  • Организация — название организации, в которой производится установка произвольного значения наработки.
  • Подразделение — наименование подразделения организации, в котором производится установка произвольного значения наработки.
  • N — номер записи об установке произвольного значения наработки объекта ремонта в списке регистрируемых значений.
  • Объект ремонта — наименование объекта ремонта, для которого регистрируется наработка.
  • Показатель — обозначение показателя, в разрезе которого фиксируется наработка объекта ремонта.
  • Единица измерения — единица измерения, в которой учитывается наработка объекта ремонта.
  • Инвентарный № — инвентарный номер объекта ремонта.
  • Технологический № — технологический номер объекта ремонта.
  • Дата последней наработки — значение определяет начало периода, за который регистрируется наработка объекта ремонта.
  • Зафиксированная наработка — значение наработки на дату выполнения операции установки произвольного значения (если ранее наработка не фиксировалась по данному показателю, то поле отображается незаполненным).
  • Наработка — значение параметра наработки с даты последней наработки до даты операции установки произвольного значения, которое будет зарегистрировано для объекта ремонта.
  • Дата и время операции — значение определяет окончание периода, за который регистрируется наработка объекта ремонта (значение и начала, и окончания периода при сбросе показаний счетчика).
  • Показания счетчика на момент сброса - значение параметра наработки до выполнения операции установки произвольного значения  (сброса счетчика).
  • Абсолютная наработка – значение параметра наработки объекта ремонта за весь период его использования.
  • Начальное показание счетчика — значение параметра наработки после выполнения операции установки произвольного значения.
  • Причина изменения наработки - причина, по которой необходимо установить произвольное значение счетчика (выполнить сброс счетчика).
  • Распространять на подчиненные ОР - при установленном значении позволяет распространить введенные данные на все объекты ремонта, находящиеся в иерархии к текущему.
  • Иерархия - структура иерархии, которую необходимо использовать при распространении данных на подчиненные объекты ремонта.
  • Ответственный — ФИО сотрудника, ответственного за операцию установки произвольного значения наработки.
  • Комментарий — дополнительная текстовая информация.
