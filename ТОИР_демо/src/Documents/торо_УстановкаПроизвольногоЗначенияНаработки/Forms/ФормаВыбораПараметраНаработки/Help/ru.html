Документ «Установка произвольного значения наработки»

Документ предназначен для установки наработки в произвольное значение всвязи с необходимостью отражения факта проведения капитального ремонта оборудования, замены неисправного счетчика наработки, разделения подсчета абсолютного значения наработки оборудования и значения отображаемого на счетчике наработки и др. Так же документ позволяет зафиксировать наработку оборудования с момента последней регистрации наработки документом "Учет параметром наработки" до момента сброса показателей счетчика.

В форме выводятся следующие поля:

  • Дата - дата регистрации документа.
  • Номер — номер документа.
  • Организация — название организации, в которой производится установка произвольного значения наработки.
  • Подразделение — наименование подразделения организации, в котором производится установка произвольного значения наработки.
  • Ответственный — ФИО сотрудника, ответственного за операцию установки произвольного значения наработки.
  • Комментарий - дополнительная текстовая информация.
