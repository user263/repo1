Документ «Настройка регистрации наработки зависимых ОР»

В форме выводятся следующие поля:

  • Номер — номер документа.
  • от — дата регистрации документа.
  • Дата начала действия коэффициентов — дата, с которой будут актуальны уточняющие коэффициенты для расчёта наработки подчинённых объектов ремонта.
  • Организация — название организации, в которой вводится документ.
  • Подразделение — наименование подразделения организации, в котором вводится документ.
  • N — номер в списке.
  • Объект ремонта — объект ремонта, для подчинённых которому настраивается регистрация наработки.
  • Ответственный — ФИО пользователя, являющегося ответственным за ввод документа.
  • Комментарий — дополнительная информация.

 

Закладка «Параметр наработки»

  • N — номер в списке.
  • Организация — название организации, которой принадлежит объект ремонта.
  • Подразделение — наименование подразделения организации, которому принадлежит объект ремонта.
  • Направление — направление использования объекта ремонта.
  • Тип объекта ремонта — принадлежность к типовому объекту ремонта.
  • Объект ремонта — объект ремонта, для которого производится пересчёт наработки.
  • Коэффициент пересчёта — значение уточняющего коэффициента.
