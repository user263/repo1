Документ «Установка коэффициентов перевода стоимостей ремонтов»

В форме выводятся следующие поля:

  • Номер — номер документа.
  • от — дата и время регистрации документа.
  • Организация — наименование организации, в которой сформирован документ.
  • N — порядковый номер записи в списке.
  • Коэффициент — тип коэффициента, с помощью которого выполняется расчет стоимостей ремонтов.
  • Организация — наименование организации, для которой задается значение коэффициента.
  • Подразделение— наименование подразделение, для которого задается значение коэффициента.
  • Направление — направление объектов ремонта.
  • Объект ремонта — объект ремонта, для которого задается коэффициент.
  • Классификатор ремонтов — вид ремонта.
  • Показатель коэффициента — значение коэффициента указанного типа.
  • Ответственный — ФИО пользователя, который создал документ. По умолчанию в поле подставляется ФИО пользователя, под которым был выполнен вход в систему.
  • Комментарий — дополнительная текстовая информация о документе установки коэффициентов перевода стоимостей ремонтов.
