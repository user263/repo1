Документ «Наряд на выполнение ремонтных работ»

Документ предназначен для регистрации начала этапа работ по каждой операции, привязанной к заявке на ремонт.

В форме выводятся следующие поля:

  • Номер — номер документа в системе.
  • от — дата и время создания документа в системе.
  • Организация – наименование организации, в которой оформлен наряд на выполнение ремонтных работ.
  • Подразделение – наименование подразделения, в котором оформлен наряд на выполнение ремонтных работ. По значению данных полей накладывается отбор при подборе ремонтов в документ (можно выбрать только объекты, принадлежащие указанным Организации и Подразделению). 
  • Ответственный - ФИО пользователя, создавшего документ.
  • Комментарий – дополнительная текстовая информация о наряде.

Закладка «Ремонтные работы»

  • Объект ремонта – наименование объекта ремонта, на ремонт которого оформлен наряд.
  • Вид ремонтных работ – вид ремонтных работ.
  • Дата начала — планируемая дата и время начала ремонта.
  • Дата окончания — планируемая дата и время окончания ремонта.
  • Г — признак того, что объект ремонта находится на гарантии.
  • Ремонтная работа – наименование ремонта или технологической операции, заявленной на выполнение.
  • Количество – количество заявленных на выполнение ремонтов или тех. операций.
Кнопка [Подбор. Ремонты] предназначена для оперативного формирования списка с помощью обработки «Подбор работ по ТОиР из документов-оснований».
 
Кнопка [Подбор. Внешние основания для работ] предназначена для оперативного формирования списка с помощью обработки "Подбор внешних оснований для работ из документов-оснований".
 
Кнопка [Подбор] предназначена для оперативного формирования списка с помощью обработки "Подбор технологических операций".
 

Закладка «Хозспособ»

  • Сотрудник – ФИО сотрудника, который является исполнителем ремонтных работ по наряду.
  • Квалификация – название квалификации сотрудника, являющегося исполнителем ремонтных работ.
  • Дата начала  — планируемая дата и время начала ремонта.
  • Дата окончания — планируемая дата и время окончания ремонта.
  • КТУ — коэффициент трудового участия.

 

Закладка «Подрядчики»

  • Контрагент – наименование организации-контрагента, выполняющей ремонтные работы по наряду.
  • Дата начала — планируемая дата и время начала ремонта.
  • Дата окончания — планируемая дата и время окончания ремонта.

 

Закладка «Документы-основания»

  • № — порядковый номер записи в списке.
  • Документ основание – наименование, номер и дата документа-основания.

 

Подзакладка «Внешние основания для работ»

  • № — порядковый номер записи в списке.
  • Объект ремонта — наименование объекта ремонта.
  • Плановая дата обработки — плановая дата работы по внешнему основанию.
  • Описание — дополнительная текстовая информация о внешнем основании для работ.
