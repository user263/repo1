Журнал запланированных ремонтов

Журнал вызывается из формы списка документа "План-график ППР" нажатием кнопки Журнал ремонтов.

Журнал предназначен для отображения информации по запланированным ремонтам. Для всех указанных нормативных ремонтов указываются следующие данные: Технологические операции ремонта, Дата ремонта, Подразделение-исполнитель ремонта, а также Описание, Время выполнения и материальные затраты конкретной технологической операции, входящей в состав ремонта.

Реквизиты:

  • Период с ... по ... - период, на который будет выведена информация по запланированным ремонтам.
  • Организация - организация-владелец объекта ремонта.
  • Подразделение - подразделение-исполнитель запланированного ремонта.
  • Объект ремонта - объект ремонта, по которому в журнал будет выведена информация по запланированным ремонтам.
