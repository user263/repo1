Документ «Внешние основания для работ»

Режим предназначен для ведения списка документов, предписывающих необходимость устранения нарушений, таких как: предписания об устранении нарушений, акты технического расследования происшествий, акты сезонного осмотра зданий и сооружений, акты диагностических испытаний либо иные сходные документы.

 

В форме выводятся следующие поля:

  • Номер — номер документа.
  • от — дата и время регистрации документа в системе.
  • Организация — наименование организации, в которой сформирован документ.
  • Подразделение — наименование, в которой сформирован документ.
  • Внешнее основание от — дата и время документа внешнего основания для работ.
  • Дата создания — дата и время документа внешнего основания для работ.
  • Ответственный — ФИО пользователя, который создал документ. По умолчанию в поле подставляется ФИО пользователя, под которым был выполнен вход в систему.
  • Комментарий — дополнительная текстовая информация о внешнем основании для работ.

 

      Закладка «Внешние основания для работ»

      • N — порядковый номер записи в списке.
      • Объект ремонта — наименование объекта ремонта.
      • Плановая дата обработки — плановая дата работы по внешнему основанию.
      • Описание — дополнительная текстовая информация о внешнем основании для работ.
      • Исполнитель — наименование исполнителя заявленных в рамках внешнего основания работ.
      • Типовой дефект — наименование часто встречающегося, типового дефекта.
      • Ремонтная работа — наименование технологической карты или технологической операции, которую необходимо выполнить для объекта ремонта в рамках внешнего основания для работ.
      • Количество — количество технологических операций или технологических карт.

       

      Закладка «Описание»

      • Дополнительная текстовая информация о внешнем основании для работ.
