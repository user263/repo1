Документ «Установка коэффициентов ремонтных особенностей»

В форме выводятся следующие поля:

  • Номер — номер документа.
  • от — дата регистрации документа.
  • Организация — название организации, в которой производится установка коэффициентов ремонтных особенностей.
  • N — номер объекта ремонта в списке.
  • Коэффициент — тип устанавливаемого коэффициента.
  • Подразделение — наименование подразделения организации, в котором производится установка коэффициентов ремонтных особенностей.
  • Направления — направление использования объекта ремонта.
  • Объект ремонта — наименование объекта ремонта, для которого устанавливается коэффициент ремонтных особенностей.
  • Классификатор ремонтов — вид ремонта, для которого устанавливаются коэффициенты.
  • Показатель коэффициента — значение коэффициента ремонтных особенностей.
  • Ответственный — ФИО сотрудника, ответственного за операцию установки коэффициентов ремонтных особенностей.
  • Комментарий — дополнительная информация.
