Документ "Регистрация значений внешних показателей"

Документ предназначен для регистрации в системе значений показателей процессов (KPI), которые нельзя получить из данных, имеющихся в ТОиР.

В табличной части документа указывается:

- элемент справочника «Показатели процессов (KPI)» с вариантом заполнения «Внешний показатель»;

- значение показателя;

- период, на который получено значение;

- аналитика (подразделение, объект ремонта).

 

Обязательность заполнения аналитики определяется типом показателя (т.е. для показателя объекта ремонта необходимо заполнить оба поля, для показателя подразделения – только подразделение, для показателя организации – ничего). Организация обязательна для всех типов показателей и вынесена в шапку.
