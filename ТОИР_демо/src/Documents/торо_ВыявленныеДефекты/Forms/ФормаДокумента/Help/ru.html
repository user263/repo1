Документ «Выявленные дефекты»

В форме выводятся следующие поля:

  • Номер — номер документа в журнале.
  • от — дата регистрации документа.
  • Дата обнаружения — дата и время обнаружения дефектов оборудования.
  • Организация – название организации, в которой зарегистрирован дефект. По умолчанию в поле подставляется организация из настроек пользователя, регистрирующего дефект в системе.
  • Подразделение – подразделение, в котором зарегистрирован дефект. По умолчанию в поле подставляется подразделение из настроек пользователя, регистрирующего дефект в системе
  • Инициатор – подразделение, обнаружившее дефект: подразделение организации пользователя, организация подрядчика и др.
  • Дефект выявлен в процессе – условие выявления дефекта.
  • Ответственный — ФИО пользователя, ответственного за регистрацию выявленного дефекта.
  • Комментарий — дополнительная текстовая информация о выявленном дефекте.

 

Закладка «Дефекты оборудования

  • Документ-основание для отбора — обозначение документа, являющегося основанием для регистрации выявленного дефекта.
  • Объект ремонта — наименование объекта ремонта, в котором обнаружен дефект.
  • Подразделение-исполнитель — подразделение, указанное у объекта ремонта в качестве ремонтного подразделения.
  • Отказавший элемент — наименование ремонтной единицы, для которой регистрируется неисправность.
  • Типовой дефект — наименование часто повторяющего дефекта в рамках определённого направления объектов ремонтных работ.
  • Вид дефекта — наименование вида дефекта.
  • Причина дефекта — название причины, вследствие которой оборудование вышло из строя.
  • Критичность дефекта — наименование критичности дефекта.
  • Описание дефекта — краткое описание дефекта.
  • Последствия дефекта — последствия возникновения дефекта.
  • Направление на устранение — описание мероприятий, направленных на устранение дефекта.
  • Г - признак нахождения оборудования на гарантии.

Кнопка [Подбор ОР] предназначена для оперативного формирования списка с помощью обработки "Подбор объектов ремонтных работ".

Кнопка [Подбор предписаний] предназначена для оперативного формирования списка с помощью обработки "Подбор внешних оснований для работ из документов-оснований".

Закладка «Документы-основания»

  • № — порядковый номер записи в списке.
  • Документ-основание — наименование документа-основания.
