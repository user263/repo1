Журнал дефектов

Журнал вызывается из формы списка документа "Выявленные дефекты" нажатием соответствующей кнопки.

Журнал предназначен для отображения информации по зарегистрированным в системе выявленным дефектам. Данные представлены в хронологическом порядке по дате и времени обнаружения дефекта. В журнале выводится следующая информация: Дата и время обнаружения дефекта, Объект ремонта, Отказавший элемент объекта ремонта, Ответственный за регистрацию дефекта, Причина, Описание, Вид и Последствия дефекта, ФИО лица, кому сообщено о дефекте, Замечания руководства и мастера, Направление на устранение и Состояние устранения дефекта

Реквизиты:

  • Период с ... по ... - период, на который будет выведена информация по выявленным дефектам. 
  • Организация - название организации, в которой зарегистрирован дефект.
  • Подразделение - название подразделения, в котором зарегистрирован дефект.
  • Объект ремонта - объект ремонта, по которому в журнал будет выведена информация по выявленным дефектам.
