Документ «Акт о выполнении регламентного мероприятия»

Документ предназначен для отражения факта выполнения мероприятий ремонтным подразделением, учитывая выполнение в разрезе каждой технологической операции.

 

В форме выводятся следующие поля:

  • Дата — дата регистрации документа.
  • Номер — номер документа.
  • Общее время выполнения — суммарное время выполнения работ по акту.
  • Ответственный — ФИО пользователя, который создал документ. По умолчанию в поле подставляется ФИО пользователя, под которым был выполнен вход в систему.
