Документ «Акт о выполнении регламентного мероприятия»

В форме выводятся следующие поля:

  • Номер — номер документа.
  • от — дата регистрации документа.
  • Организация — наименование организации, в которой формируется акт о выполнении регламентного мероприятия.
  • Подразделение — наименование подразделения, которое является основным исполнителем по документу.
  • Общее время выполнения – суммарное время выполнения работ по акту.
  • Завершить мероприятия – признак, указывающий на то, что выполнение заявленных регламентных мероприятий завершено.
  • Ответственный — ФИО пользователя, который создал документ. По умолчанию в поле подставляется ФИО пользователя, под которым был выполнен вход в систему.
  • Комментарий – дополнительная текстовая информация об акте.

 

Закладка «Ремонтные работы»

  • Документ-основание для отбора – наименование, номер и дата документа, на основании которого сформирован акт.
  • N – порядковый номер записи в списке.
  • Список объектов — наименование маршрута, на ремонт которого оформлен акт.
  • Вид мероприятия — вид ремонтных работ.
  • Дата начала — планируемая дата и время начала мероприятия.
  • Дата окончания — планируемая дата и время окончания мероприятия.
  • Документ источник — номер и дата документа-основания для выполнения регламентных мероприятий по акту.
  • Завершить ремонт — признак завершения мероприятия.
  • Ремонтная работа — наименование ремонтной работы.
  • Выполнено — признак выполнения ремонтной работы.

Кнопка [Подбор] предназначена для оперативного формирования списка с помощью обработки «Подбор работ по ТОиР из документов-оснований».

Подзакладка «Ремонтные работы»/«Материальные затраты»

  • Номенклатура – наименование номенклатуры.
  • Характеристика – характеристика номенклатуры.
  • Ед. изм. – единица измерения номенклатуры.
  • Количество – количество номенклатуры.
  • Сумма – стоимость номенклатуры.

 

Подзакладка «Ремонтные работы»/«Исполнители»

Хозспособ

  • Сотрудник – сотрудник, который является исполнителем тех. операции.
  • Квалификация – квалификация сотрудника-исполнителя технологической операции.
  • Дата начала – дата и время начала выполнения работ.
  • Дата окончания – дата и время окончания работ.
  • Время работы – время выполнения работы сотрудником.
  • КТУ – коэффициент трудового участия.
  • Сумма к начислению – сумма для начисления сотруднику за выполнение ремонтной операции.

Для равномерного распределения суммы, начисленной сотрудникам за выполнение технологических операций предусмотрена кнопка [Расчитать по КТУ]. Таким образом, при нажатии кнопки система автоматически распределит суммы исполнителям в соответствии со значением в поле "КТУ".

Гипперссылка «Ввести сдельный наряд» предназначена для ввода документа «Сдельный наряд», реквизиты которого будут автоматически заполнены в соответствии с документ-основанием.

Подрядчики

  • Контрагент – наименование контрагента, выполнившего ремонтную работу.
  • Договор контрагента – название договора с контрагентом.
  • Дата начала – фактическая дата начала ремонтных работ.
  • Дата окончания – фактическая дата окончания ремонтных работ.
  • Сумма – общая сумма раемонта, проведённого контрагентом.

 

Закладка «Документы-основания»

  • N — номер элемента списка.
  • Документ основание — обозначение наряда, являющегося документом-основанием.
  • Закрыть наряд — признак закрытия наряда на регламентное мероприятие, являющееся документом основанием.
