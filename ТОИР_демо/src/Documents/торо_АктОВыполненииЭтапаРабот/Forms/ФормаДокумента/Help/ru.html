Документ «Акт о выполнении этапа работ»

В форме выводятся следующие поля:

  • Номер — номер документа.
  • от — дата регистрации документа.
  • Организация — наименование организации, в которой формируется акт о выполнении этапа работ.
  • Подразделение — наименование подразделения, которое является основным исполнителем по акту о выполнении этапа работ.
  • Общее время выполнения – время выполнения работ по акту.
  • Завершить ремонтные работы – признак, указывающий на то, что выполнение заявленных ранее ремонтных работ завершено
  • Ответственный — ФИО пользователя, ответственного за оформление заявки.
  • Комментарий – дополнительная текстовая информация об акте.

 

Закладка «Ремонтные работы»

  • N – порядковый номер записи в списке.
  • Объект ремонта – наименование объекта ремонта, на ремонт которого оформлен акт.
  • Вид ремонтных работ – вид ремонтных работ.
  • Дата начала  — фактическая дата и время начала ремонта.
  • Дата окончания  — фактическая дата и время окончания ремонта.
  • Заверш. рем. — признак завершения ремонта.
  • Г — признак гарантийного ремонта.
  • Ремонтная работа — наименование выполненного ремонта или технологической операции.
  • Количество — количество заявленных на выполнение ремонтов или тех. операций.
  • % выполнения — уровень выполнения ремонтной работы в процентах..
Кнопка [Подбор. Ремонты] предназначена для оперативного формирования списка с помощью обработки «Подбор работ по ТОиР из документов-оснований».
 
Кнопка [Подбор. Внешние основания для работ] предназначена для оперативного формирования списка с помощью обработки "Подбор внешних оснований для работ из документов-оснований".

 

Для заполнения материалов для работ по нормативам технологических операций предназначена команда [Заполнить. Заполнить материалы по нормам].

Для заполнения материалов для работ по смете ремонта предназначена команда [Заполнить. Заполнить материалы по плановым затратам].

Кнопка [Подбор] предназначена для оперативного формирования списка с помощью обработки "Подбор технологических операций".
 
Гипперссылка «Ввести акт об оказании производственных услуг» предназначена для ввода документа с соответствующим названием по текущему акту.

 

Подзакладка «Ремонтные работы»/«Материальные затраты»

  • Номенклатура – наименование номенклатуры.
  • Ед. изм. – единица измерения номенклатуры.
  • Количество – количество номенклатуры.

 

Подзакладка «Ремонтные работы»/«Запчасти»

  • N – порядковый номер записи в списке.
  • Наименование старой запчасти – наименование номенклатуры заменённой запчасти.
  • Серия старой запчасти – серийный номер замененной запчасти. Поле заполняется в случае, если для номенклатуры задана необходимость вести учет по сериям, и такой складской учет действительно ведется.
  • Статус движения старой запчасти – способ замены запчасти.
  • Наименование новой запчасти – наименование номенклатуры устанавливаемой запчасти.
  • Серия новой запчасти – серийный номер устанавливаемой запчасти. Поле заполняется в случае, если для номенклатуры задана необходимость вести учет по сериям, и такой складской учет действительно ведется.
  • Статус движения старой запчасти – способ замены запчасти.
  • Ед. изм. – единицы измерения запчасти.
  • Кол-во – количество замененных запчастей.

Кнопка [Заполнить по заявке] предусматривает автоматическое заполнеие списка номенклатурой из заявки на ремонт.

Подзакладка «Внешние основания для работ»

  • Объект ремонта — наименование объекта ремонта.
  • Плановая дата ремонта — плановая дата работы по внешнему основанию.
  • Описание — дополнительная текстовая информация о внешнем основании для работ.

 

Подзакладка «Ремонтные работы»/«Исполнители»

Хозспособ

  • Сотрудник – сотрудник, который является исполнителем тех. операции.
  • Квалификация – квалификация сотрудника-исполнителя технологической операции.
  • Дата начала – дата и время начала выполнения работ.
  • Дата окончания – дата и время окончания работ.
  • Время работы – время выполнения работы сотрудником.
  • КТУ – коэффициент трудового участия.
  • Сумма к начислению – сумма для начисления сотруднику за выполнение ремонтной операции.

Подрядный

  • Контрагент – наименование контрагента, выполнившего ремонтную работу.
  • Договор – название договора с контрагентом.
  • Дата начала – фактическая дата начала ремонтных работ.
  • Дата окончания – фактическая дата окончания ремонтных работ.
  • Сумма – общая сумма раемонта, проведённого контрагентом.

Закладка «Документы-основания»

  • N – номер элемента списка.
  • Документы-основания – обозначение наряда, являющегося документом-основанием.
