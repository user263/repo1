Документ «Акт о выполнении этапа работ»

Документ предназначен для отражения факта выполнения работ ремонтным подразделением, учитывая процент выполнения в разрезе каждой технологической операции. Одним актом фиксируется выполнение работ по одному объекту ремонта.

 

В форме выводятся следующие поля:

  • Дата — дата регистрации документа.
  • Номер — номер документа.
  • Общее время выполнения – время выполнения работ по акту.
  • Ответственный — ФИО пользователя, ответственного за оформление заявки.
