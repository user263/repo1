Документ «Оценка работы подрядчика»

В форме выводятся следующие поля:

  • Номер — номер документа.
  • от — дата и время регистрации документа в системе.
  • Организация — наименование организации, в которой сформирован документ.
  • Подразделение – наименование подразделения организации, в котором сформирован документ.
  • Акт о выполнении работ — номер и дата акта о выполнении работ подрядчиком.
  • Контрагент — наименование контрагента, выполнявшего ремонт.
  • Оценка для всех работ — признак того, что оценка необходимо установить для всех работ выбранного акта о выполнении этапа работ. Если признак не установлен, то оценку необходимо установить для каждой работы, указанной в акте о выполнении этапа работ, при этом в левой части формы отображается специальная табличная часть с перечнем работ из акта.
      • N — сумма документа установки лимита.
      • Объект ремонта — объект ремонта, для которого задается коэффициент.
      • Вид ремонта — вид ремонта.
      • Вид оценки — наименование вида оценки.
      • Оценка — оценка работы подрядчика по десятибальной шкале. Поле заполняется автоматически как среднее арифметическое оценок по заданным в нижней правой табличной части критериям.
      • Критерий оценки — критерий, по которому оценивается работа подрядчиков.
      • Значение — оценка работы подрядчика по десятибальной шкале по указанному критерию.
      • Ответственный — ФИО пользователя, создавшего документ.
      • Комментарий — дополнительная информация о документе.
