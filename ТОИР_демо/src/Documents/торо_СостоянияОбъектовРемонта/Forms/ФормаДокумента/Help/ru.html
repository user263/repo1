Документ «Состояния объектов ремонта»

Список предназначен для ввода документов о простоях и испытаниях оборудования.

 

В форме выводятся следующие поля:

  • Номер — номер документа.

  • от — дата регистрации документа.

  • Организация — наименование организации, в которой вводится документ.

  • Подразделение – наименование подразделения, в котором регистрируется документ.

  • Вид эксплуатации – вид эксплуатации объекта ремонта.
  • Ответственный – ФИО пользователя, являющегося ответственным за регистрацию изменения состояния оборудования.
  • Комментарий — дополнительная информация о документе присвоения.

 

[Операция. Состояние на дату]

  • N — порядковый номер записи.

  • Объект ремонта — наименование объекта ремонта, для которого регистрируется изменение состояния его эксплуатации.

  • Распространять на подчиненные ОР — признак того, что состояние объектов ремонта, входящих в состав вводимого объекта ремонта, будет также изменено.

  • Иерархия — структура иерархии, в которой рассматривается данный объект ремонта.

  • Дата изменения состояния — дата, на которую необходимо ввести изменение состояние оборудования.
  • Вид эксплуатации – вид эксплуатации объекта ремонта с даты изменения состояния.
  • Вид пуска – вид пуска объекта ремонта.
  • Примечание – дополнительная текстовая информация об изменении состояния оборудования.

 

[Операция. Состояние на период]

  • N — порядковый номер записи.

  • Объект ремонта — наименование объекта ремонта, для которого регистрируется изменение состояния его эксплуатации.

  • Распространять на подчиненные ОР — признак того, что состояние объектов ремонта, входящих в состав вводимого объекта ремонта, будет также изменено.

  • Иерархия — структура иерархии, в которой рассматривается данный объект ремонта.

  • Начало периода — дата начала периода, на который необходимо ввести состояние оборудования.

  • Вид эксплуатации на период – вид эксплуатации объекта ремонта в рамках периода.

  • Окончание периода — дата окончания периода, на который необходимо ввести состояние оборудования.

  • Вид эксплуатации — вид эксплуатации объекта ремонта по окончании периода.

  • Вид пуска – вид пуска объекта ремонта.

  • Примечание — дополнительная текстовая информация об изменении состояния оборудования.

     

    Кнопка  [Подбор]  предназначена для оперативного формирования списка с помощью обработки «Подбор объектов ремонта».

    С помощью кнопки  [Заполнить по умолчанию] осуществляется заполнение состояний введенных в табличную часть объектов ремонтов на основании данных, уже имеющихся в системе документах по состоянию оборудования.
