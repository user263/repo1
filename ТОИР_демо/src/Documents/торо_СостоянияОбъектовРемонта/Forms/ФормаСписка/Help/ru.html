Документ «Состояния объектов ремонта»

Список предназначен для ввода документов о простоях и испытаниях оборудования.

 

Кнопка [Создать. Создать документ на дату] предназначена для ввода состояния оборудования, если известна только дата начала состояния.

Кнопка [Операция. Создать документ на период] предназначена для  ввода состояния оборудования, если известен весь период состояния.

 

В форме выводятся следующие поля:

  • Дата — дата регистрации документа.

  • Номер — номер документа.

  • Комментарий — дополнительная информация о документе присвоения.
