Документ «Учет контролируемых показателей»

В форме выводятся следующие поля:

  • Номер — номер документа.
  • от — дата регистрации документа.
  • Организация – наименование организации, которая производит контроль измеряемых показателей.
  • Подразделение – наименование подразделения организации, которое производит контроль измеряемых показателей.
  • Ответственный – ФИО пользователя, который является ответственным за регистрацию контролируемых показателей.
  • Комментарий — дополнительная текстовая информация о контролируемых показателях.

 

Раздел «Оборудование»

  • Оборудование — название объекта ремонта, для которого регистрируется значения контролируемых показателей.

Кнопка [Подбор] предназначена для оперативного формирования списка с помощью обработки "Подбор объектов ремонта".

 

Раздел «Контролируемые  показатели»

  • Показатель — наименование контролируемого показателя.
  • Ед. изм. – единица измерения контролируемого показателя.
  • Значение — значение контролируемого показателя.

Кнопка [Заполнить] предназначена для автоматического заполнения табличной части контролируемых показателей.
