Документ «Выбытие объекта ремонта»

В форме выводятся следующие поля:

  • Номер — номер документа.
  • Дата — дата регистрации документа.
  • Организация — название организации, в которой фиксируется выбытие объекта ремонта.
  • Склад — склад для списываемого объекта ремонта.
  • Подразделение — наименование подразделения организации, в котором фиксируется выбытие объекта ремонта.
  • Объект ремонта — списываемый объект ремонта.
  • Дата списания — дата, с которой объект ремонта считается списанным.

 

Закладка «Общие сведения»

  • Инвентарный № — инвентарный номер объекта ремонта.
  • Дата ввода в эксплуатацию — дата ввода в эксплуатацию объекта ремонта.
  • Входил в состав — вышестоящий объект ремонта. Имеется возможность менять иерархию. 
  • Ответственный — ФИО пользователя, являющегося ответственным за ввод исходных данных.
  • Комментарий — дополнительная информация о воде первоначальных данных.

     

    Закладка «Номенклатура»

  • N — номер по порядку.
  • Номенклатура — наименование номенклатуры.
  • Характеристика — характеристика номенклатуры.
  • Количество — количество единиц номенклатуры.
  • Склад — слад принадлежности номенклатуры.

 

Закладка «Комиссия»

  • N — номер по порядку.
  • Сотрудник — ФИО сотрудника, входящего в состав комиссии по списыванию объекта ремонта.

 

Закладка «Список подчинённых»

  • N — номер по порядку.
  • Объект ремонта — объект ремонта, подчинённый списываемому.
