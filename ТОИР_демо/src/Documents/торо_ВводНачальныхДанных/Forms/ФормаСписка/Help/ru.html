Документ «Ввод начальных данных»

Документ предназначен для ввода начальных данных для объектов ремонта. По каждому объекту ремонта, включенному в документ можно задать значения измеряемых показателей, информацию о последних ремонтных работах, а также сведения о значениях показателей наработки. Первоначальные данные вводятся в систему в процессе ее внедрения на этапе вода НСИ.

В форме выводятся следующие поля:

  • Дата — дата регистрации документа.
  • Номер — номер документа.

 

Кнопка [Создать. Документ по объекту ремонта] позволяет ввести начальные данные для отдельных объектов ремонта.

Кнопка [Создать. Документ по списку объектов ремонта] позволяет ввести данные для списка объектов регламентных мероприятий
