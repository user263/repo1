Документ «Ввод начальных данных»

В форме выводятся следующие поля:

[Создать. Документ по объекту ремонта]

  • Номер — номер документа.
  • от — дата регистрации документа.
  • Организация — название организации, в разрезе которой вводятся первоначальные данные.
  • Подразделение — наименование подразделения организации, регистрирующей документ.
  • N – порядковый номер записи в списке.
  • Положение —  текущее положение вводимого объекта ремонта.
  • Объект ремонта — оборудование, для которого производится ввод начальных данных.
  • Инвентарный номер — инвентарный номер оборудования.
  • Распространять на подчиненные ОР — при установленном значении позволяет распространить введенные данные на все объекты ремонта, находящиеся в иерархии к текущему.
  • Иерархия —  структура иерархии, которую необходимо использовать при распространении данных на подчиненные объекты ремонта.
  • Начальное состояние — вид эксплуатации объекта ремонта.
  • Ответственный — ФИО пользователя, являющегося ответственным за ввод исходных данных.
  • Комментарий — дополнительная информация о воде первоначальных данных.

Кнопка [Подбор] предназначена для оперативного формирования списка с помощью обработки "Подбор объектов ремонтных работ".

Закладка «Выполненные работы»

  • Вид ремонтных работ — ссылка на последний ремонт, который проводился для оборудования.
  • Дата окончания ремонта — дата и время окончания последнего ремонта, который проводился для оборудования.

 

Закладка «Показатели»

  • Дата контроля — дата и время проведения контроля измеряемых показателей оборудования.
  • Показатель — название измеряемого показателя.
  • Значение — регистрируемое значение измеряемого показателя.

 

Закладка «Наработка»

  • Показатель — название показателя наработки.
  • Дата работы по — дата регистрации значения показателя наработки оборудования.
  • Значение — регистрируемое значение показателя наработки оборудования.

 

Закладка «Запчасти»

  • Номенклатура – наименование номенклатуры.
  • Серийный номер – серийный номер запчасти.
  • Статус движения ЗЧ – вид замены запчасти. Значение в поле выбирается из встроенного списка и носит справочное значение.
  • Кол-во – количество указанных запчастей в оборудовании.

 

Закладка «Пуски объекта ремонта»

  • Вид пуска – вид пуска оборудования.
  • Количество – количество пусков указанного вида.

 

[Создать. Документ по списку объектов ремонта]

  • Номер — номер документа.
  • от — дата регистрации документа.
  • Организация — название организации, в разрезе которой вводятся первоначальные данные.
  • Подразделение — наименование подразделения организации, регистрирующей документ.
  • N – порядковый номер записи в списке.
  • Список объектов – наименование списка объектов регламентных меропрятий.
  • Вид мероприятия – вид последнего мероприятия, проведенного для списка объектов регламентных мероприятий.

  • Дата последнего мероприятия – дата и время окончания последнего меропрятия.

  • Ответственный — ФИО пользователя, являющегося ответственным за ввод исходных данных.
  • Комментарий — дополнительная информация о воде первоначальных данных.
