Справочник «Организации»

В форме выводятся следующие поля:

  • Код - код организации.
  • Наименование - краткое название организации.
  • Префикс - символьное значение, которое используется для нумерации документов, регистрируемых в разрезе этой организации (если значение не указано, то нумерация будет сквозная).
  • ИНН – идентификационный номер налогоплательщика.
  • КПП - код причины постановки на учёт.
  • Юридический адрес - юридический адрес организации.
  • Фактический адрес - фактический адрес организации.
  • Телефон - номер телефона организации.
  • Факс - номер факса организации.
  • Электронная почта - электронная почта организации.
  • Почтовый адрес - полный почтовый адрес организации.
  • Другое - прочая информация об организации.
