Роли контактных лиц партнеров

Справочник Роли контактных лиц содержит список возможных ролей контактных лиц по отношению к клиенту или поставщику, контактным лицом которого они являются. Контактное лицо, например, может выполнять роль менеджера по закупкам, курьера, секретаря и т.д. В дальнейшем эта информация может быть использована для принятия решения, к какому из контактных лиц клиента обращаться по какому вопросу.

Важно. Информацию о ролях контактных лиц можно использовать только в том случае, если включена соответствующая функциональная опция (Администрирование/CRM и продажи).

Как добавить новую роль контактного лица в список ролей контактных лиц?
  • В списке ролей контактных лиц нажмите на кнопку Создать.
  • Будет создана карточка новой роли контактного лица.
