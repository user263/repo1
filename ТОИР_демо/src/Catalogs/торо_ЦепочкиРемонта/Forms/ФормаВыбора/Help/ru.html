Справочник «Цепочки ремонтов»

Справочник предназначен для создания фиксированных ремонтных циклов.

В форме выводятся следующие поля:

  • Код — код цепочки ремонтов.
  • Наименование — название цепочки ремонтов.
  • Способ учета даты отсчета – название способа, в соответствии с которым учитывается дата отсчета графика ППР.
