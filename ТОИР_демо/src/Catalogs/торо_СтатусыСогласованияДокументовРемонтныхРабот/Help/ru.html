Справочник «Статусы согласования документов ремонтных работ»

Форма вызывается из пункта меню «Справочники» → «Классификаторы» → «Статусы согласования документов ремонтных работ»

Справочник предназначен для ведения списка возможных статусов документов системы.

Реквизиты:

  • Код — код статуса документа.
  • Наименование — наименование статуса документа.
