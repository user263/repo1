Единицы измерения

Справочник Единицы измерения предназначен для ведения списка всех единиц измерения, определенных для позиций номенклатуры (товаров или услуг). Каждая позиция характеризуется базовой единицей измерения. При необходимости номенклатуре могут быть назначены упаковки - альтернативные единицы измерения, для каждой из которых определяется коэффициент пересчета относительно базовой единицы.

Как добавить новую единицу измерения в список?

Как добавить новую единицу измерения в список?

  • В списке единиц измерения нажмите на кнопку Создать.
  • Будет открыта карточка новой единицы измерения.
  • В карточке единицы измерения заполните данные о коде единицы измерения, полное и краткое наименование, международное сокращение.
  • При заполнении данных используйте данные Общероссийского классификатора единиц измерения.
  • После ввода данных нажмите на кнопку Записать и закрыть.

Важно. Для ввода новой единицы измерения в соответствии с классификатором используйте команду Подобрать из классификатора.
