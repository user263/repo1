Справочник «Технологические карты ремонтов»

Справочник предназначен для хранения ремонтов, зарегистрированных в системе. Каждый ремонт представляет собой технологическую карту, состоящую из перечня технологических операций и нормативных затрат на их выполнение.

В форме выводятся следующие поля:

  • О.О. — признак опасной работы.
  • П.О. — признак работы повышенной опасности.
  • Наименование — краткое название технологической карты.
  • Код - код технологической карты ремонта.
  • Норма времени — длительность выполнения работ по данной технологической карте.
