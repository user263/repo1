Справочник «Способы согласования документов»

Форма вызывается из пункта меню «Справочники»→ «Классификаторы»→ «Способы согласования документов»

Справочник предназначен для хранения правил, по которым изменяется статус документа "Заявка на выполнение ремонтных работ"

 

Реквизиты:

  • Код — код правила в справочнике.

  • Наименование — название способа согласования документов.

  • Организация – наименование организации, для которой используются правила согласования.

  • Табличная часть формы Матрица перехода предназначена для задания правил согласования документов.
  • Статус документа – наименование статуса документа.
  • Начальный – признак того, что статус является начальным для документа.
  • Согласование завершено – признак конечного статуса для документа.
  • Проводить документ – признак того, что документ в статусе может быть проведен.
  • Шаблон E-Mail – наименование шаблона письма, которое будет выслано в качестве уведомления ответственным за изменение статуса документа пользователям.

 

Закладка «Согласование»
  • Пользователь - список пользователей, определяющий порядок согласования документов в выделенном статусе.
  • Замещающий - список пользователей, которые могут согласовать документ вместо пользователя.

Закладка «Оповещение»

  • Оповещаемое лицо - содержит список сотрудников или контактных лиц , которым будет выслано оповещение об изменении статуса документа
  • E-Mail - адрес электронной почты, на который  будет выслано оповещение об изменении статуса документа
