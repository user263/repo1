Справочник «Приоритеты объектов ремонта»

Справочник содержит приоритеты объектов ремонта, которые используются в системе для организации строгой последовательности выполнения работ по ремонту и обслуживанию оборудования, а в частности при расчете графика ППР.

 

В форме выводятся следующие поля:

  • Наименование — обозначение приоритета объекта ремонта.
  • Код — код приоритета объектов ремонта в системе.
