Справочник «Технологические операции»

В форме выводятся следующие поля:

  • Родитель — группа, в которую входит технологическая операция
  • Наименование — название технологической операции.
  • Код — код технологической операции.
  • Опасная — признак опасной работы.
  • Работа повышенной опасности — признак работы повышенной опасности.
  • Единица измерения — единица измерения технологической операции.
  • Коэффициент —
  • Расценка — стоимость выполнения операции.
  • Расценка в валюте учёта — стоимость выполнения операции в учётной валюте.
  • Норма времени — время, требуемое на выполнение операции.

 

Закладка «Материальные затраты»

  • N — номер единицы номеклатуры в списке.
  • Номенклатура — наименование позиции номенклатуры, которая входит в перечень материально-технического обеспечения для технологической операции.
  • Упаковка, Ед. изм. — значение из классификатора единиц измерений в упаковке.
  • Качество — качество номенклатуры. Выбирается из списка: Новый, Ограничено годен, Не годен.
  • Количество упаковок — количество упаковок, необходимых для выполнения операции.
  • Количество единиц — количество единиц номенклатуры с учётом количества упаковок, необходимых для выполнения операции.

 

Закладка «Трудовые затраты»

  • N — номер по порядку.
  • Квалификация — наименование квалификации исполнителя ремонтной операции.
  • Количество — количество исполнителей ремонтной операции.
  • Время работы – длительность выполнения ремонтной операции исполнителем указанной квалификации.

 

Закладка «Инструменты и техника»

  • Инструмент — наименование инструмента или единицы техники, требуемой для исполнения норматива по операции ремонта.
  • Ед. изм. — обозначение единицы измерения инструмента или единицы техники.
  • Количество — количество единиц указанного инструмента или техники, требуемое для соблюдения норматива.
