Справочник «Списки объектов регламентных мероприятий»

Справочник предназначен для хранения списков объектов регламентных мероприятий.

Элемент справочника(маршрут) - это список оборудования, объединенного по принципу единого расписания выполнения мероприятий ТОиР, с указанием списка доступных мероприятий ТОиР (осмотров, проверок, поверок, диагностических измерений и т.п.), назначенных для выполнения.

В форме выводятся следующие поля:

  • Наименование — название списка объектов регламентного мероприятия в справочнике.
  • Код — код списка объектов регламентного мероприятия.
