Справочник «Виды коэффициентов ремонтных особенностей»

В форме выводятся следующие поля:

  • Код – код вида коэффициента ремонтных особенностей.
  • Наименование – наименование вида коэффициента ремонтных особенностей.
  • Вид коэффициента 
    • устанавливаемый - значения коэффициента для каждого оборудования устанавливает сам пользователь;
    • рассчитываемый - значения коэффициента рассчитывается системой самостоятельно.
      • Коэффициент зависит от 
        • Возраста оборудования - в основу расчета коэффициента ложится возраст объекта ремонта.
          • Считать возраст оборудования от: Даты ввода в эксплуатацию - возраст оборудования считается от даты ввода в эксплуатацию.
          • Считать возраст оборудования от: Даты выпуска - возраст оборудования считается от даты изготовления.
          • Учитывать график работы оборудования - определяет необходимость использовать график работы объекта ремонта при расчете значения коэффициента.
        • От значения наработки - в основу расчета коэффициента ложится абсолютная наработка объекта ремонта.
          • Параметр наработки - значение параметра наработки, по которому будет рассчитываться коэффициент.
          • Учитывать простои оборудования - определяет необходимость учитывать простои оборудования при расчете плановой наработки.
