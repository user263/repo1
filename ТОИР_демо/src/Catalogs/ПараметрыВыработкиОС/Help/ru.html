Справочник «Параметры наработки ОР»

Справочник предназначен для хранения списка параметров, по которым в системе учитывается наработка оборудования. Заполнение элементов справочника выполняется непосредственно в табличной части списка.

В форме выводятся следующие поля:

  • Наименование - краткое наименование.
  • Код - код параметра.
  • Единица измерения - наименование единицы измерения, в которой учитывается наработка ОР для параметра (значение выбирается из классификатора единиц измерения).
