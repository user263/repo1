Справочник «Показатели процессов (KPI)»

В справочнике настраиваются источники данных для вывода в монитор показателей (KPI). Возможны 3 варианта:

  • Схема компоновки данных, настроенная по определенным правилам. Редактирование СКД возможно только в режиме запуска «Толстый клиент (управляемое приложение)». В тонком клиенте доступен только просмотр текста запроса.
  • Внешний показатель. В этом случае данные вводятся в систему с помощью документа «Регистрация значений внешних показателей».
  • Формула от других показателей (поиск показателей выполняется по реквизиту «Представление для формул»).

Реквизит «Тип показателя» определяет аналитические разрезы, по которым получаются данные. Самый низкий уровень, «Показатель объекта ремонта», включает организацию, подразделение и объект ремонта. Самый высокий уровень – «Показатель организации», включает только организацию.

Также в справочнике указывается название отчета, используемого для расшифровки показателя. Можно оставить поле пустым, тогда будет использоваться форма расшифровки по умолчанию, отображающая таблицу данных показателя до всех преобразований, выполняемых перед выводом в монитор.
