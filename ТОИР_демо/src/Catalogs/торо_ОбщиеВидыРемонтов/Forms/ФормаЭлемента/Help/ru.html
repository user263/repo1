Общие виды ремонтов

Справочник Общие виды ремонтов предназначается для классификации Видов ремонтов, определенных для класса объектов эксплуатации.
Для элемента справочника можно определить статью расходов, по которой будут отражаться все выполняемые типовые ремонты, связанные с данным общим видом.
